Task List API

Task List is a web API built to organize your thoughts. Through it you can create, read, update, and delete tasks and goals. Additional features include sorting tasks and goals, marking tasks and goals complete, receiving feedback on completed tasks via Slack, and more.

Technologies used: Python, Flask, SQL, SQLAlchemy, Postgres, Postman, and Heroku.

Swap Items Meet

The backend for an app where vendors can swap items like clothing, electronics, and more between different inventories. Project creates many class definitions, their attributes and instance methods, along with other cool features.

Technologies used: Python, unit tests, code coverage

Word Game

This game was a pair programming project. An anagram is a word or phrase formed by rearranging the letters of a different word or phrase. In this game, a player is given a random set of letters and must make an anagram using those letters. Each submitted word will score points.

Technologies used: Python, pair programming
